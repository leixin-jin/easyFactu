# Features Index (Template)

使用方法
- 新建功能文档：复制模板到 features 目录
  - `cp doc/agents/template/features/feature.md doc/agents/features/<slug>.md`
- 然后在本文件下方追加一行链接到新文件。

示例（创建后请改为你自己的链接）
- [等位功能](doc/agents/features/waitlist.md) – 客户加入/叫号/过号流程
- [餐桌管理](doc/agents/features/table-management.md) – 开台/并台/转台/清台
